## 项目结构

- `src/`：源代码

## 快速开始

```shell
git clone --depth=1 https://github.com/Qnurye/data-warehouse-and-data-mining.git
cd data-warehouse-and-data-mining
python -m venv venv
source venv/bin/activate # Windows 系统请参考 https://wiki.archlinux.org/title/Installation_guide
pip install -e .
```
